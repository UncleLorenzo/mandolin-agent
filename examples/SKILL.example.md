# Ship a launch post

## When to use
The owner is announcing something new — a product, a drop, a milestone — and wants it written
the way *they* would write it, not the way an AI would.

## Procedure
1. Load the Signature. Read Voice and Standards before writing a single word.
2. Draft the post in that voice. If it could've come from anyone, it isn't done.
3. Check it against Standards. Cut anything that hypes, over-explains, or begs.
4. Hand it back as a draft, not a decree. The owner ships, not the agent.
